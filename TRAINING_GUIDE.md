- Try different browser

- Ensure good lighting
- Hold phone steady
- Try manual entry

**Photos Won't Upload:**
- Check internet connection
- Verify file size (<10MB)
- Check file format (JPG, PNG)
- Try smaller batch
- Clear browser cache

**Page Not Loading:**
- Check internet connection
- Refresh page (F5)
- Clear browser cache
- Try different browser
- Check if service is down
